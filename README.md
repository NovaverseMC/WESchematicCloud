# WESchematicCloud

## Description
A [WorldEdit](https://github.com/EngineHub/WorldEdit) addon to implement the `//download` and `/schem load url:` commands from FAWE.

## Permissions
- Same permission as FAWE

## Commands
- Same schematic download/load commands as FAWE
